LB-4 Використання Socket API для мережевих додатків
Інструменти

1. Підготовлена та встановлена IDE для розробки на python, наприклад pycharm.
2. Аккаунт на github для завантаження виконаного завдання
Завдання

Створити клієнт та сервер з використанням Socket API згідно із завданнями:
1. [easy] Розробити echo-сервер з використанням python socket API, який
приймає з'єднання від клієнта та повертає клієнту отримані дані. Приклад
реалізації сервера - https://docs.python.org/3/library/socket.html#example
 
a. [easy] Розробити echo-клієнт з використанням python socket API, який
встановлює з'єднання з сервером, надсилає строкові дані та отримує
відповідь. Приклад реалізації клієнта -
https://docs.python.org/3/library/socket.html#example
2. [medium] Модифікувати TCP-сервер з п.1, щоб він працював постійно та
обробляв запити від багатьох клієнтів послідовно. Для тестування сервера.
запустити декілька разів клієнт з пункту 1-а.

3. [medium/hard] Розробити TCP-сервер з використанням python socket API,
який приймає текстовий файл від клієнта та зберігає його локально на стороні
сервера.

a. [medium/hard] Розробити TCP-клієнт, який відправляє будь-який
текстовый файл на сервер
